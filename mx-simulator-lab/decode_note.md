## 2. 할인 분류 — "두 얼굴" (★파일 구현 기준, 정책서와 다름)

| 처리 | 해당 항목 | 손익 타이밍 | 메커니즘 |
|---|---|---|---|
| **매출차감** | **포인트류(구독포인트·다품목 포인트) 뿐** | **당월** 즉시 | 당월 손익에 비용으로 바로 |
| **에누리** | **선납할인 + 단품(구독료할인) + 결합(다품목·추가구독·추가구매·재구독)** = 할인 전부 | **구독기간 스프레드** | 매출인식기준 구독료↓ → 이자수익↓ → 기간 전체 분산 |

> ※ 정책서 프로세스표엔 "다품목"이 매출차감에 적혀 있으나, 그건 **다품목 *포인트*** 이야기.
> **파일에서 다품목/단품/추가구독/추가구매/재구독 *할인*은 전부 에누리(DD열)로 모여 매출인식기준 구독료에서 차감**된다. **파일이 governing.**

### "결합할인"은 우산 용어
```
결합할인 = 다품목 + 추가구독 + 추가구매 (+재구독)   ← 한 단어로 퉁침
단품할인 = 구독료할인 (단일 제품)                    ← 결합과 별개
선납할인 = 선납 비율에 대한 할인                      ← 선납금(미리 낸 돈)과는 다름
```

## 3. 할인 6단계 패턴 (선납할인·단품·결합 전부 동일)

각 시트 좌측(B27~B48 등)에 `(계산용)→(최종)`, `V+/V-`로 이 패턴이 반복된다.
```
① 할인금(v+)   = 기준구독료(v+) × 할인율
② 할인금(v-)   = ① ÷ 1.1
③ 월할인금(v-) = ROUND/ROUNDUP(② ÷ 개월, -2)   ← 월로 쪼개 100원 단위
④ 월할인금(v+) = ③ × 1.1
⑤ 최종할인금   = ③(또는 ④) × 개월               ← 다시 ×기간 = 깔끔한 총액
⑥ 반영 구독료  = 구독료 − 월할인금
```
> 왜 빙 도나? `총액×율`을 바로 월로 나누면 끝자리가 지저분 → **월 단위 먼저 반올림 후 ×기간**으로 매월 청구액을 딱 떨어지게.
> **단수조정(A안/B안)·소수점 차이는 잡이익/잡손실 처리** 이슈로 연결됨.

### 선납금(≠선납할인) 주의
- **선납금** = 고객이 미리 낸 목돈 (예: 30%). 구독료를 선납분만큼 미리 차감.
- **선납할인** = 선납했다고 깎아주는 할인 (예: 4%). 이건 에누리.
- 둘은 별개 라인. (CE 예: 선납금 30% = 3,528,000 / 선납할인 4% = 475,200)

## 4. 엔진 — 리스 상각표 (금융리스 채권)

**구독 = 제품을 리스(할부판매)한 것.** 채권을 매월 이자수익만큼 늘리고 리스료수령만큼 줄여 만기 0으로 상각.

```
컬럼: 기초 → 이자수익 → 리스료수령(−) → 기말
 기말ₜ = 기초ₜ + 이자수익ₜ − 리스료수령ₜ
 이자수익ₜ = 기초ₜ × 월이자율
 월이자율 = (연이자율)/12,  연이자율은 "채권 PV = 리스료 합 PV"가 되도록 역산(IRR)
```
실측(선납+단품+결합 엔진): 리스채권 4,065,091 / 총 순수구독료 5,172,000 / 연이자율 **9.91%** / 월 0.826% / 리스료수령 86,200·월(첫달 75,100 일할).

> **핵심**: 할인(에누리)으로 순수구독료가 줄면 → 채권·리스료가 줄고 → **연이자율(IRR)을 다시 푼다**. 이게 "할인이 이자수익을 줄여 기간에 스프레드"되는 실제 메커니즘.

## 5. 이벤트별 정산 로직 (시트 우측 블록)

공통 헤더(B4~B16): 품목·구독기간·혜택가·총구독료·월구독료. 그 아래 선납/할인 6단계. 우측에 회차표 + 이벤트 정산.

### 정상만기 (1,2,16)
- 회차별 `구독료청구` (제품+케어+보증). **첫달·막달 일할**(사용일수/그달일수).

### 중도해지 (4,5,7,8)
정산 = **위약금 + 사용분 청구 + 선납잔액 환불 + 등록비 청구 + 할인금 회수**
- 위약금: **비과세 → 총구독료(v+) 기준**
- 기준: **일수비율** (위약금·할인회수·선납환불·등록비 전부)
- 할인금 회수 = 받은 할인을 잔여기간만큼 토해냄(clawback)

### 중도완납 (10)
- **중도완납금 + 사용분 청구 − 선납액 차감**
- 청구 기준: **케어+보증 제외한 매출인식기준구독료의 일수비율**

### 연체미납 강제해지 (11,12,13)
- 연체 **1회→2회→3회** 누적, **누적 연체이자**(가산이자, 익월 청구일부터)
- **연체 3개월 초과 → 강제해지**: 위약금 + 사용분 + 연체이자 + 선납잔액 환불 + 등록비
- 기준 시점: **계약중지일** (변형: 연체3회차 발생월 전월말일 — 숨김시트)

### 할인원복 (9) — 결합 깨짐
- 결합(다품목) 조건이 깨지면(예: 묶음 중 1제품 해지) **다품목 할인 회수**
- **익익월**부터 구독료 원복 (예: 21회차 구독료부터 정상가 적용)
- "차감성 할인 원복" — 에누리로 깎아준 걸 되돌림

### AS미사용 기간 보상 (14,15)
- AS 미사용 기간만큼 **고객에게 보상금 환급** (ROUNDUP)
- **기본 구독료 기준** — 할인여부 무관

## 6. 학습 순서 (의존성 기반)

```
[엔진] 17 베이스 상각표 → 18 +선납 → 3 +전체할인     ← 여기서 IRR 재계산 체득
   │
[정상] 1/2 정상만기(CE/MX) → 16 첫달·막달 일할         ← 회차표·일할 체득
   │
[해지] 4 중도해지(선납) → 7 +단품 → 8 +결합            ← 정산 5종 체득
   │
[기타] 10 중도완납 → 11~13 연체강제해지 → 9 할인원복 → 14/15 AS보상
```

> 각 칸: ① 내가 빈 시트에 재현 → ② 원본과 대조 → ③ 막히면 정답 로직 확인.

## 7. 미해결/주의 (작업 시 챙길 것)
- **단수조정**: 월할인금 100원 단위 처리 → 총할인금 차이 → 잡이익/잡손실
- **CE vs MX**: 파손보험 비과세 → v+ 계산·할인기준 차이
- **위약금·연체이자**: 비과세 → v+ 기준 계산
- **할인금 회수(clawback)**: 중도해지·할인원복의 핵심. 일수비율로 잔여 할인 회수

# Part A. 중도완납 (시트10)

## 1. 개념
고객이 **남은 기간 구독료를 한 번에 미리 완납**하고 끝내는 것. 중도'해지'와 달리 위약금이 아니라 **남은 값을 정상가로 완납**(단, 케어·보증 제외 + 선납액 차감).

## 2. 핵심 수식 (중도완납 확정일 R24, 경과 16일)
```
경과일 S24 = R24 − G24 = 16
미사용일 T24 = (H24−G24+1) − S24 = 14        (당월 중 안 쓴 날)
이후 월 T25~ = 그달 전체일수

① 중도완납금 U24 = ROUNDDOWN( 매출인식기준구독료(C13) × Σ미사용일/전체일(C8), −2 )
                = ROUNDDOWN(10,540,800 × Σ(T24:T77)/2191, −2) = 7,740,800
   · 케어+보증 제외한 "매출인식기준구독료"의 미사용기간분을 한 번에

② 사용분 청구 V24 = ROUNDDOWN( 선납반영 월구독료(N24) × 당월경과일/그달, −2 ) = 57,400

③ 선납액 차감 W24 = −ROUNDUP( 선납금(C26) × 잔여비율, −2 ) = −2,590,900
```
> 메모: **중도완납 청구 기준 = 케어+보증 제외한 매출인식기준구독료의 일수비율.** 위약금은 비과세 v+ 기준.

## ✅ 체크포인트(완납)
1. 완납금 모수? → *매출인식기준구독료(케어·보증 제외)*
2. 미사용일 정의? → *전체일 − 경과일*
3. 선납액은? → *잔여비율로 차감(환급 성격)*

# Part B. 연체미납 강제해지 (시트11)

## 1. 개념
고객이 안 내면 → **연체이자 가산** → 3개월(3회) 초과 시 **강제해지**, 그동안 밀린 것 + 위약금까지 한 번에 정산.

## 2. 연체이자 누적 (가산 6%, 익월 청구일부터, 일할 365)
```
연체 1회 U = ROUNDDOWN( (밀린구독료 N×1.1) × 6% × (다음청구−이번청구−1)/365, −1 )
연체 2회 V, 3회 W = 그 다음달들 각각 추가 가산
누적 연체이자 T = SUM(U:W)

예시 흐름:
  20회차: 미납 발생 (107,700)
  21회차 R = N21 + T21 = 107,700 + 580 = 108,280   (가산 580 붙음)
  22회차 R = 109,340   (누적 1,640)
  23회차 R = 111,130   (누적 3,430)
```
> 메모: **연체 발생 이후 익월 청구일부터 가산이자.** 연체이자는 비과세 → v+ 기준.

## 3. 강제해지 정산 (계약중지 X24 = 마지막 청구일 + 2일)
```
경과일 Y, ΣY = SUM(Y5:Y24)

위약금     Z24 = ROUNDDOWN( 총구독료v+(C16) × 잔여비율 × 20%, −1 ) = 1,899,560
사용분     AA24 = Σ(밀린 미납분 AA20:AA23) + 당월일할 = 488,200
할인금 회수 AB24 = ROUNDDOWN( 할인(C30) × 경과비율, −2 ) = 126,200
연체이자   AC24 = 누적 T = 3,430
선납잔액 환불 AD24 = −ROUNDUP( 선납금(C26) × 잔여비율, −2 ) = −2,590,900
등록비     AE24 = ROUNDDOWN( 등록비(C40)/1.1 × 경과비율, −2 ) = 24,100
```
> 중도해지(M3)와 골격은 같고 **추가로 (1) 누적 연체이자 (2) 밀린 미납분 합산**이 붙는다.

## 4. 변형 시트
- **12. _MX**: 파손보험 비과세 처리
- **13. +선납+단품+결합(CE)**: 할인회수에 단품·결합까지
- (숨김) **연체3회차 발생월 전월말일 기준**: 중지 시점 기준만 다름

## ✅ 체크포인트(연체)
1. 가산이자율·기준일수? → *6%, 365일 일할*
2. 가산 시작 시점? → *연체 익월 청구일부터*
3. 강제해지 트리거? → *연체 3개월(3회) 초과*
4. 사용분에 무엇이 더해지나? → *밀린 미납분 합산*
5. 위약금/연체이자 과세? → *둘 다 비과세 v+ 기준*

## ⚠️ 함정
- 연체이자를 단순 월 6%로 ×하면 틀림 → **일수(365) 일할**
- 강제해지 사용분에서 **밀린 미납 합산 누락**
- 중도완납 모수에 케어·보증 포함시키는 실수

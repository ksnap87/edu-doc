# M2 (D3–4) — 정상만기 청구 + 할인 6단계

> 대상 시트: **1. 정상만기_선납+단품+결합(CE)** · 2. (MX) · **16. 첫달·막달 일할**
> 정답지: `answer_key.py` (할인·49회차 청구)

---

## 1. 큰 그림

정상만기 시트 = **왼쪽(할인 계산) + 오른쪽(회차별 청구표)**.
- 왼쪽 B10~C58: 구독료에서 **선납금·선납할인·단품·결합**을 차례로 깎아 "반영 구독료"를 만든다 (할인 6단계).
- 오른쪽 I~U: 그 구독료를 **매월 청구**(첫달/막달은 일할). 3가지 시나리오를 나란히 보여준다.

---

## 2. 할인 6단계 패턴 (★선납할인·단품·결합 전부 동일)

같은 패턴이 B27~ 반복. **단품 할인(5%) 예시**(CE):
```
① 할인금(v+)   = 기준구독료(v+) × 5%          = 646,668
② 할인금(v-)   = ① ÷ 1.1                       = 587,880
③ 월할인금(v-) = ROUND(② ÷ 72, −2)            = 8,200   ← 월로 쪼개 100원
④ 월할인금(v+) = ③ × 1.1                       = 9,020
⑤ 최종할인금   = ③ × 72                        = 590,400 ← 다시 ×기간 = 깔끔한 총액
⑥ 반영 구독료  = 구독료 − 월할인금
```
> **왜 빙 도나?** `총액×율`을 바로 72로 나누면 끝자리가 지저분 → **월 단위로 먼저 반올림 후 ×72** 하면 매월 청구액이 딱 떨어진다. 이때 생기는 **총액 미세차이는 잡이익/잡손실**(단수조정 이슈).

**선납금(≠선납할인)** — 미리 낸 목돈:
```
선납금 % = 30%
(최종) 선납금(V-) = 3,528,000,  월 선납금(V-) = 49,000
→ 매월 구독료에서 선납분만큼 미리 차감
```
**할인 누적 결과(CE 실측)**: 기본 월구독료 163,300 → 선납반영 107,700 → +단품 → +결합 → 최종 **92,000**(R열).

## 3. 오른쪽 청구표 (3 시나리오 병렬)

| 블록 | 열 | 내용 |
|---|---|---|
| 기본 (v-) | J,K,L,M | 구독료청구·제품·케어·보증 |
| 선납+선납할인 (v-) | N,O,P,Q | 〃 (선납 반영) |
| 선납+선납할인+단품+결합 (v-) | R,S,T,U | 〃 (전체 할인) |

**핵심 수식**
```
청구일      I = EOMONTH(결제월,0)+15           (익월 15일)
1회차(일할) J5 = ROUNDDOWN((C14/C7)×(H5−G5+1)/DAY(H5), −2) = 103,400
2회차~      J6 = C15 (월구독료) = 163,300
  제품 K = C13/C7 = 146,400,  케어 L = C11/C7 = 15,100,  보증 M = C12/C7 = 1,800
  제품 = 구독료 − 케어 − 보증 (차액금)
선납시      N6 = C33 = 107,700
전체할인    R6 = C57 = 92,000
```
- **첫달**: 사용일수(H5−G5+1)/그달일수 만큼 일할
- **케어·보증은 할인 안 함** → 모든 시나리오에서 L,M 동일(P=L, Q=M, T=L)
- **고객노출 v+**: W/X열. 선납시 최종납부금 X7 = 기본 − 선납월할 − 선납할인월할 − 단품월할

## 4. 시트16 — 첫달·막달 일할 (별도 정리)

첫달·막달의 일할을 따로 검증하는 시트. 핵심:
```
1회차 J4 = ROUNDDOWN((C13/C6)×(사용일/그달일), −2) = 103,400
2회차~ J5 = C14 = 163,300,  제품 K = C12/C6 = 146,400
v+ N = J×1.1 = 179,630
* 제품 = 구독료일할 − 케어일할 − 보증일할 (차액금)
* "1회차 수식을 그대로 적용해도 같은 값" (메모)
```
> 막달(49회차) = 계약 마지막 짜투리. **1회차 + 막달 = 한 달치**가 되도록 일할.

## 5. CE vs MX 차이 (반드시 외울 2가지)
```
(1) 파손보험 비과세 → v+ 계산이 다름
    [CE] 총구독료(v+) = 총구독료(v-) × 1.1
    [MX] 총구독료(v+) = (제품+보증+케어)×1.1 + 파손보험   ← 파손만 비과세
(2) 할인기준금액 = 파손보험 제외
    할인기준(v+) = 총구독료(v+) − 파손보험   (파손엔 할인 0%)
```

## ✍️ 직접 작성 연습
1. 단품 5% 할인을 **6단계**로 손계산 → 8,200 / 590,400 나오는지
2. 청구표 1행(1회차 일할) + 2행(정상월) 만들고 아래 복사
3. 선납 30% 넣어 N열(107,700)·R열(92,000) 재현
4. MX 모델로 (1)(2) 차이 적용해보기

## ✅ 체크포인트
1. 할인을 왜 월로 쪼갰다 다시 ×기간? → *끝자리 정리(단수조정)*
2. 케어·보증 할인율은? → *0% (할인 대상 아님)*
3. 청구일 공식? → *EOMONTH(결제월,0)+15*
4. 제품 청구액 구하는 법? → *구독료 − 케어 − 보증 (차액)*
5. MX에서 파손보험은 v+ 계산 시? → *×1.1 안 함, 비과세로 그대로*

## ⚠️ 함정
- 선납**금**(목돈)과 선납**할인**(4%)을 혼동 — 별개 라인
- 1회차 일할 빠뜨림
- 단수조정 차이를 0으로 가정 → 총액 안 맞음
- MX 파손보험 ×1.1 잘못 적용

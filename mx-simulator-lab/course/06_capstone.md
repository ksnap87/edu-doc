# M6 (D12–14) — 통합 복습 + 직접작성 시험

> 목표: 빈 엑셀에서 **새 모델로 처음부터** 정상만기·중도해지·연체·완납·원복·AS보상을 스스로 작성.

---

## 1. 전체 한 장 요약 (눈 감고 그릴 수 있어야)

```
[입력] 혜택가·케어·보증·구독기간·할인율·선납%
   │
   ▼
[엔진] 리스채권 = MIN(제품v-, PV) (−선납)
        리스료 = 순수구독료/기간
        IRR(F3) = RATE(기간, −리스료, 채권)        ← 할인 바뀌면 재계산
        상각표: 기말 = 기초 + 기초×IRR − 리스료
   │
   ▼
[할인] 6단계: 총액×율 → ÷1.1 → 월반올림 → ×1.1 → ×기간 → 차감
        에누리(선납·단품·결합) = 매출인식기준 구독료↓ → 이자수익↓ → 기간 스프레드
        매출차감(포인트류) = 당월 손익
   │
   ▼
[청구] 청구일=익월15일, 1·막달 일할, 제품=구독료−케어−보증
   │
   ▼
[이벤트 정산]
   중도해지   : 위약금(v+×잔여×20%) + 사용분 + 할인회수(경과) + 선납환불(잔여) + 등록비
   중도완납   : 완납금(매출인식기준×미사용/전체) + 사용분 − 선납차감
   연체해지   : +누적연체이자(6%,365일할) +밀린미납 합산, 3회 초과 강제
   할인원복   : 결합깨짐 → 다품목 회수(경과) + 익익월 정상가
   AS보상     : 기본구독료×미사용일 (회사→고객, ROUNDUP)
```

## 2. 비율 치트시트 (★자주 헷갈림)

| 정산 | 비율 | 방향 |
|---|---|---|
| 위약금 | 잔여비율 × 20% | 고객→회사 |
| 사용분 | 당월 경과일/그달 | 고객→회사 |
| 할인금 회수 | **경과**비율 | 고객→회사 |
| 선납잔액 환불 | **잔여**비율 | 회사→고객(−) |
| 등록비 | 경과비율 | 고객→회사 |
| 중도완납금 | 미사용/전체 | 고객→회사 |
| AS보상 | AS 미사용일 | 회사→고객(−) |

> 경과비율 = Σ경과일/2191, 잔여비율 = (2191−Σ경과일)/2191

## 3. 과세 치트시트
- **×1.1 (과세)**: 구독료·제품·케어·보증·등록비·사용분
- **비과세 (v+ 그대로)**: 파손보험·위약금·연체이자·분실료·파손료

## 4. 단수조정 치트시트
- 구독료·사용분·할인: `ROUNDDOWN(...,−2)` (100원 내림)
- 위약금·분실/파손료·연체이자: `−1` (10원)
- 고객에게 주는 것(AS보상·선납환불): `ROUNDUP` (고객 유리)

## 5. 졸업 시험 (스스로 채점)

**시험 A — 엔진**: 임의 모델(혜택가 X, 기간 48개월)로 리스채권·IRR·상각표 3행 작성.
→ 채점: 마지막 회차 기말이 0에 수렴하는가?

**시험 B — 정상청구**: 선납30%+단품5%+결합5% 반영 월구독료와 1회차 일할.
→ 채점: 케어·보증은 할인 0인가? 1회차+막달=한달치인가?

**시험 C — 중도해지**: 24개월차 해지 시 위약금·할인회수·선납환불.
→ 채점: 할인회수=경과, 선납환불=잔여 방향 맞나? 위약금 비과세인가?

**시험 D — 연체**: 3개월 연체 후 강제해지. 누적 연체이자 + 밀린미납.
→ 채점: 가산이자 365일할인가? 미납 합산했나?

**시험 E — 원복/보상**: 결합 깨짐 원복 시점, AS보상 모수.
→ 채점: 익익월인가? 보상이 기본구독료 기준인가?

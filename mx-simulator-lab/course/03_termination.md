# M3 (D5–7) — 중도해지 정산

> 대상 시트: 4. 중도해지_선납,선납할인 · 5. (MX) · 7. (+단품) · **8. 중도해지_선납+단품+결합할인**
> 8번이 가장 완전한 형태 → 이걸로 마스터하면 4·5·7은 부분집합.

---

## 1. 큰 그림 — 해지하면 정산할 5가지

고객이 중간에 그만두면, **제품판정일** 기준으로 5가지를 한 번에 계산한다:

| 정산 항목 | 방향 | 기준 | 한 줄 의미 |
|---|---|---|---|
| **위약금** | 고객→회사 | 잔여비율 | 약속 깬 대가 |
| **사용분 청구** | 고객→회사 | 당월 일할 | 마지막 달 쓴 만큼 |
| **할인금 회수** | 고객→회사 | 경과비율 | 받은 할인 토해냄(clawback) |
| **선납잔액 환불** | 회사→고객(−) | 잔여비율 | 미리 낸 돈 돌려줌 |
| **등록비 청구** | 고객→회사 | 경과비율 | 초기 등록비 |

> 핵심 두 비율: **경과비율** = Σ경과일/전체일(2,191), **잔여비율** = (전체일−Σ경과일)/전체일.

## 2. 핵심 수식 (시트8, 제품판정일 V24 = 2027-04-17, 경과일 합계 기준)

```
경과일 W = 해당월 일수(H−G+1), 마지막달은 V24−G24 = 16일
ΣW(경과누적) = SUM(W5:W24)

① 위약금       X24 = ROUNDDOWN( 총구독료v+(C16) × 잔여비율 × 20%, −1 )
                   = ROUNDDOWN(12,933,360 × (2191−ΣW)/2191 × 20%, −1) = 1,899,560
   · 비과세 → v+ 기준 그대로 (AC24 = X24)

② 사용분 청구   Y24 = ROUNDDOWN( 반영월구독료(R24) × 당월경과일/그달일수, −2 ) = 49,000
   · v+ : AD24 = Y24 × 1.1

③ 할인금 회수   Z24 = Σ[ ROUNDDOWN(각 할인 × 경과비율, −2) ]
                   = 선납할인(C31)·결합(C51)·단품(C41) 각각 일할 = 426,400
   · 경과한 기간만큼 받은 할인을 회수

④ 선납잔액 환불 AA24 = −ROUNDUP( 선납금(C22) × 잔여비율, −2 ) = −2,590,900
   · 음수 = 고객에게 돌려줌

⑤ 등록비 청구   AB24 = ROUNDDOWN( 100,000/1.1 × 경과비율, −2 ) = 24,100

[v+ 환산] AD~AG = ×1.1  (단 위약금 AC는 비과세 그대로)
```

> ⚠️ 기준 명시(시트 메모): **위약금·할인회수·선납환불·등록비 = 일수비율**, 사용분 = 제품판정일 **전날까지** 사용.

## 3. 분실료·파손료 (비과세, 혜택가 v+ 기준)
```
분실료 AJ12 = ROUNDDOWN( 혜택가v+(C10×1.1) × 잔여비율, −1 ) = 5,603,220
파손료 AK12 = ROUNDDOWN( 분실료 × 50%, −1 )                 = 2,801,610
```
> 제품을 잃어버리거나 망가뜨린 경우 추가 청구. 비과세라 v+ 기준.

## 4. 엔진 연결
중도해지는 M1 엔진의 **채권 잔액**과 연결된다(시트6 결합·A해지·월할이 그 처리). 해지 시점의 미상각 채권 = 회사가 아직 못 받은 원금 → 위약금·정산으로 메움.

---

## 5. 4·5·7번과의 차이 (부분집합)
- **4. 선납,선납할인**: 할인회수 Z에 단품·결합이 없음(선납할인만)
- **5. _MX**: 파손보험 비과세 처리 추가
- **7. +단품**: Z에 단품 추가, 결합 없음
- **8. +결합**: 전부 (= 위 본문)

---

## ✍️ 직접 작성 연습
1. 경과비율·잔여비율을 ΣW로 정의
2. 위약금 X = C16 × 잔여 × 20% → 1,899,560 재현
3. 할인회수 Z = 선납할인+단품+결합 각 일할 → 426,400
4. 선납환불 AA = −선납금 × 잔여 → −2,590,900
5. v+ 환산(×1.1) 하되 위약금은 제외

## ✅ 체크포인트
1. 위약율? → *총구독료 v+의 20%*
2. 할인금 회수는 어느 비율? → *경과비율(받은 만큼 토해냄)*
3. 선납환불 부호가 음수인 이유? → *회사가 고객에게 지급*
4. 위약금이 비과세인 이유로 어떤 기준? → *v+ 기준 그대로(×1.1 안 함)*
5. 사용분은 며칠까지? → *제품판정일 전날까지*

## ⚠️ 함정
- 할인회수에 **잔여비율**을 쓰면 안 됨 → **경과비율**
- 선납환불은 **잔여비율** (반대)
- 위약금·분실/파손료에 ×1.1 잘못 적용
- 단수조정 자리: 위약금 −1(10원), 사용분 −2(100원), 분실료 −1

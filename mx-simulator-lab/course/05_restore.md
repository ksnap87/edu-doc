# Part A. 할인원복 (시트9) — 결합이 깨졌을 때

## 1. 개념
**결합(다품목) 할인은 "여러 개 묶어 산" 조건**으로 받은 것. 묶음 중 하나를 해지하면 조건이 깨지므로 → **다품목 할인을 회수하고, 구독료를 정상가로 원복**한다. 단, 즉시가 아니라 **익익월**(다다음 달)부터 반영.

## 2. 시나리오 3단 (열 블록)
| 단계 | 열 | 상태 |
|---|---|---|
| (기존) 선납+선납할인+다품목 | R,S,T,U | 결합 할인 적용 중 |
| (변경) 선납+선납할인 유지, **다품목 원복** | V,W,X,Y | 결합만 빠짐 |
| 다품목 할인 원복 시 / 원복 후 중도해지 | Z~ | 정산 |

## 3. 핵심 로직
```
· 구독료 원복은 21회차 구독료 금액부터 적용 (메모: "익익월")
· 원복 후 월구독료 V25~ = N25 (선납+선납할인만 반영, 결합 빠진 값) = 107,700
  (결합 적용 중 R열은 더 낮았음 → 원복으로 올라감)

[원복 정산 — 결합해지일 Z, 경과일 AA]
사용분 청구 AB = 결합 적용분 일할
할인금 회수 AC = 그동안 받은 다품목 할인 × 경과비율 (회수)

[원복 후 또 중도해지하면] AD~ : 위약금·사용분·할인회수·선납환불 = M3와 동일 골격
```
> 포인트: **"차감성 할인(에누리)을 되돌리는"** 케이스. 이미 깎아준 걸 경과분만큼 회수 + 앞으로는 정상가.

## ✅ 체크포인트(원복)
1. 왜 원복하나? → *결합 조건(묶음) 깨짐*
2. 언제부터 정상가? → *익익월(21회차부터)*
3. 회수 기준 비율? → *경과비율*
4. 원복 후 구독료는 어느 값? → *선납+선납할인만 반영(N열), 결합 제외*

# Part B. AS 미사용 기간 보상 (시트14)

## 1. 개념
케어/AS 서비스를 **약속한 만큼 제공 못 한 기간**이 있으면, 그만큼 **고객에게 보상금을 돌려준다**. 회사→고객 방향(환급).

## 2. 핵심 수식 (AS기간 R, 경과 S)
```
경과일 S23 = H23 − R23 + 1 = 30,  S24 = R24 − G24 + 1 = 5

미사용기간분 T23 = ROUNDUP( 기본구독료(J23) × S23/그달일수, −2 ) = 158,100
              T24 = ROUNDUP( J24 × S24/DAY(H24), −2 ) = 27,300
최종 보상금 V = T (그대로)
누적 보상 W23 = SUM(T23:T24) = 185,400

[검증용] X23 = ROUNDUP( 총구독료v-(C14) × (S23+S24)/전체일(C8), −2 ) = 187,900

[v+ 분해, X6~] 미사용기간분에서 케어·보증·구독원가 일할 차감
  Y = X − (케어일할 Z + 보증일할 AA + 구독원가일할 AB)
```
> 메모(중요): **보상은 "기본 구독료 기준", 할인여부 무관.** 즉 할인 받았어도 보상은 정상 기본구독료로 계산. 고객에게 주는 돈은 **ROUNDUP**(올림, 고객 유리).

## ✅ 체크포인트(AS보상)
1. 보상 방향? → *회사 → 고객(환급)*
2. 보상 모수? → *기본 구독료(할인 무관)*
3. 반올림 방향? → *ROUNDUP(고객 유리)*
4. 일할 기준? → *AS 미사용 기간 일수*
